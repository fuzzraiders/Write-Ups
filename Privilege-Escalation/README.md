# ⚡ Privilege Escalation

---

## 📖 About

The **Privilege Escalation** section focuses on identifying and exploiting misconfigurations that allow attackers to elevate privileges on a system.

This domain includes structured write-ups that explore how local systems are misconfigured, how access controls fail, and how attackers can gain higher privileges in real-world environments.

Content emphasizes **clarity, technical depth, and reproducibility**, ensuring each write-up delivers both practical and educational value.

---

## 🎯 Purpose

This section exists to:

* Document structured privilege escalation methodologies
* Share real-world Linux and Windows escalation techniques
* Provide hands-on learning references for local system exploitation
* Build strong foundations in post-exploitation scenarios
* Support advanced paths like red teaming and system compromise

The focus is on understanding **how privilege boundaries are broken**, not just running automated tools.

## 📚 Content Scope

Write-ups in this section may include:

* Linux privilege escalation (SUID, sudo misconfigurations, cron jobs)
* Windows privilege escalation (services, registry, token abuse)
* Misconfigured permissions and file access issues
* Environment-based escalation (PATH hijacking, writable files)
* Credential discovery and local abuse techniques
* Manual enumeration and exploitation workflows

All content is educational, practical, and responsibly documented.
