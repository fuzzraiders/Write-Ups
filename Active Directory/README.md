# 🖥️ Active Directory

---

## 📖 About

The **Active Directory** section focuses on in-depth analysis of **Windows domain environments**, covering enumeration, authentication mechanisms, misconfigurations, and domain-level attack techniques.

This domain includes structured write-ups that explore how Active Directory operates in enterprise environments, how weaknesses arise, and how attackers can leverage them in real-world scenarios.

Content emphasizes **clarity, technical depth, and reproducibility**, ensuring each write-up delivers both practical and educational value.

---

## 🎯 Purpose

This section exists to:

* Document structured Active Directory attack methodologies
* Share real-world domain exploitation techniques
* Provide hands-on learning references for internal network security
* Build strong foundations in AD enumeration and domain analysis
* Support advanced paths like red teaming and enterprise attacks

The focus is on understanding **domain behavior and attack paths**, not isolated privilege escalation techniques.

## 📚 Content Scope

Write-ups in this section may include:

* Domain enumeration (users, groups, policies, trusts)
* Kerberos-based attacks (AS-REP roasting, Kerberoasting)
* ACL abuse and permission misconfigurations
* Group Policy Object (GPO) abuse
* Domain trust exploitation
* Authentication and authorization weaknesses
* Real-world Active Directory attack paths

All content is educational, practical, and responsibly documented.
